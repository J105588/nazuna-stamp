# なずな祭街歩きスタンプラリー

なずな祭を楽しみながら街を巡る、Webベースのスタンプラリーアプリケーションです。

## 特徴
- **QRコードスキャン**: モバイルブラウザのカメラを使用してスタンプを取得
- **GPS連携**: 目的地付近にいることを確認してスタンプを有効化
- **ブラウザガード**: アプリ内ブラウザ（LINE/Instagram等）での誤動作を防止
- **コンプリート特典**: 全てのスタンプを集めると特典引き換え画面を表示

## 技術スタック
- React 19
- Vite 6
- Lucide React (アイコン)
- Html5-QRCode (スキャナー)
- Vanilla CSS (Glassmorphismデザイン)

## 開発環境の実行
```bash
npm install
npm run dev
```
